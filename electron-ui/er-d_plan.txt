# Database
## User table
- User email
- Full Name
- Password
- User type (Existing user, new user, admin) (enum type)
- Phone Number
- Membership status (Standard, Pro, Pro+, No Membership) (enum)

## Parking
- Foreign key to user
- License plate
- Start Timestamp
- End Timestamp
- Payment (boolean)

## schedule
